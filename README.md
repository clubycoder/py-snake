# Snake Game in Python
Simple terminal Snake game in Python.

* Grab hearts
* Don't run in to yourself or a wall
* Enjoy the random levels

```bash
python3 game.py
```

* `[up]`, `[down]`, `[left]`, `[right]` to move
* `[space]` to toggle pause
* `[q]` to quit
